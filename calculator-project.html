Currency Calculator

Java OOP application named Curro with real-time exchange rates from a public currency API.

This project kickstarted my Java programming skills, it is a currency conversion calculator that allows users to convert between different world currencies using real-time exchange rates retrieved from an external API. The application focuses on clean object-oriented design, accuracy, and usability.

Overview

  • Allows users to convert an amount from one currency to another.
  • Fetches up-to-date exchange rates from a public currency API.
  • Handles invalid input and API errors gracefully.

Challenges & Solutions

  • API integration: Learning how to make HTTP requests in Java and parse JSON responses into usable objects.
  • Error handling: Implemented validation for user input and fallback messaging when the API is unavailable.
  • Accuracy: Ensured conversion results are precise and properly formatted for readability.

What I Learned

  • How to integrate external services into a Java application.
  • How to design modular and maintainable object-oriented code.
  • How to debug network-related and data-format issues.
  • Technologies Java, HTTP Requests, JSON Parsing, Currency API, Git/GitHub
